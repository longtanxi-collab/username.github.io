# 观相小馆 · AI 全维面相解析

一个纯静态的中国传统面相文化娱乐体验网页。页面支持上传正脸照片，在本地浏览器中进行人脸关键点识别或结构估算，并生成包含脸型原型、三庭五眼、六维短卡和完整深度报告的面相文化解读。

## 本地运行

直接打开 `index.html` 即可预览。为了让浏览器更稳定地加载模型和脚本，也可以在项目目录启动一个静态服务器：

```bash
npx serve .
```

然后访问终端输出的本地地址。

## 部署到 Netlify

1. 登录 Netlify。
2. 选择 Add new site。
3. 上传本项目整个文件夹，或连接 Git 仓库。
4. Build command 留空。
5. Publish directory 填写项目根目录。
6. 发布后入口文件为 `index.html`。

## 部署到 GitHub Pages

1. 新建 GitHub 仓库并上传本项目文件。
2. 进入仓库 Settings。
3. 打开 Pages。
4. Source 选择主分支。
5. 目录选择 `/root`。
6. 保存后访问 GitHub Pages 生成的地址。

## 隐私说明

用户照片默认只在本地浏览器中处理，不上传服务器，不保存用户照片。刷新页面后，照片预览和分析结果会清空。如果未来加入服务器识别、云端 AI API 或账号系统，需要重新明确告知用户并获得同意。
